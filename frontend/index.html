One script boots a local Substrate node, deploys the ink! contract, launches the dashboard, and starts the orchestrator with eight parallel proving workers.

Aggregato runs against a local Portaldot-compatible development environment. The dashboard names the active endpoint so it's always clear where the demo is running.
